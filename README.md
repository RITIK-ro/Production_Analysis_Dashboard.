# Production Analysis Dashboard (Excel)

## Project Overview
This project focuses on analyzing production data using Microsoft Excel. 
The dashboard provides insights into total production quantity, production cost, 
and performance of different product types.

## Dataset Information
The dataset includes the following columns:
- Product Type
- Production Quantity
- Cost per Unit
- Total Cost
- Production Date

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization

## Key Analysis
- Total production quantity by product type
- Total production cost analysis
- Comparison of production performance across product categories

## Dashboard Features
- Interactive Pivot Table analysis
- Product type performance visualization
- Production cost insights

## Skills Demonstrated
- Data Cleaning
- Data Analysis using Excel
- Pivot Table Analysis
- Dashboard Creation
